# Structure Profile — LinkedIn Posts

## Post Length
- Target: 800–1500 characters (the LinkedIn sweet spot for engagement).
- Never exceed 3000 characters. Shorter is better — every sentence must earn its place.

## Hook (First 2 Lines)
- The first 2 lines appear before the "...see more" fold. They must grab attention.
- Use a bold claim, a surprising stat, a contrarian take, or a direct question.
- Do NOT start with "I'm excited to share..." or any generic opener.

## Paragraph Structure
- Short paragraphs: 1–3 sentences each.
- One idea per paragraph.
- Leave a blank line between every paragraph for readability on mobile.
- Instead of long paragraphs, group lines with the same idea via Line Feeds (LFs), to avoid adding spaces.

## Formatting
- **Bold** key phrases for scannability (2–4 bold phrases per post).
- No H1/H2/H3 headers — LinkedIn does not render markdown headers.
- No code blocks — LinkedIn renders them as plain text.
- Use line breaks generously. Dense walls of text kill engagement.

## Lists
- Use brief bullet points only for key takeaways or steps (max 5 items).
- Each bullet: one line, no sub-bullets.
- Prefer flowing prose over lists when possible.

## Call-to-Action (CTA)
- End with a question or call-to-action to drive comments.
- Examples: "What's your take?", "Have you tried this?", "Drop your experience below."

## Hashtags
- Do not use any hashtags. Zero tolerance for using hashtags.

## References
- Do NOT include inline citations or reference sections.
- If referencing a source, mention it naturally: "According to the Anthropic docs..." or "A recent Google DeepMind paper showed..."
- Links go in the first comment, not the post body.
