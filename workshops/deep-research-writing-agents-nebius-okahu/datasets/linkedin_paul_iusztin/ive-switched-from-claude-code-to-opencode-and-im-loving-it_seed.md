# Research Topic: Comparing AI Coding Assistants: Claude Code vs. OpenCode

This research aims to analyze and compare two specific AI coding assistants, Claude Code and OpenCode, focusing on their underlying agent orchestration, user experience, and open-source aspects, particularly in the context of a Claude Pro subscription.

## Key Questions

1. What are the core features and target users for both Claude Code and OpenCode as AI coding assistants?
2. How do the agent orchestration methodologies of OpenCode and Claude Code differ, and what impact do these differences have on task complexity handling and depth of idea understanding?
3. What are the advantages and disadvantages of OpenCode being an open-source project compared to proprietary alternatives like Claude Code?
4. How does the terminal user interface of OpenCode compare in functionality and user satisfaction to that of Claude Code?
5. What is the perceived value and performance of both tools when integrated with a Claude Pro subscription?
